# NTU Timetable Planner

# Features:
- Automated Data Scraping (Scrap from NTU Official Websites everyday at 3am SGT
- Exporting to Calender/Text
- Clash Free Timetable Generation

# Tech Stack:
- React
- WebWorkers
- Mantine UI/CSS
- Python
- Github

# Setup:
cd frontend
npm install
npm run dev

# Updating of Course Data (Manually):
cd scraper
pip install -r requirements.txt
python run_scraper.py
